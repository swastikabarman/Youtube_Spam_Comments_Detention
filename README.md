# YouTube Spam Comment Detector

A Flutter-based web application that detects potentially fake or spam comments on YouTube videos using NLP-based analysis and the YouTube Data API.

## Features

- Fetches comments from any YouTube video
- Detects potentially fake or spam comments
- Displays analysis results with confidence scores
- Clean and interactive Flutter UI
- Uses YouTube Data API for real-time comments

## Tech Stack

- Flutter
- Dart
- YouTube Data API v3
- HTTP package
